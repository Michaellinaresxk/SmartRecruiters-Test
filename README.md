- [x] Project setup with HTML, SCSS and TypeScript
- [ ] Responsive gallery component based on Figma designs
- [ ] Interactive modal with embedded YouTube video autoplay
- [ ] Modal close button stops video playback
- [ ] Keyboard navigation support with proper ARIA attributes
- [ ] Cross-browser compatibility (Chrome, Firefox, Safari, Edge)
- [ ] Unit tests for component behavior and user interactions
